# Stopwatch Web Application

A responsive stopwatch web application built using HTML, CSS, and JavaScript. The application allows users to accurately measure time with start, pause, reset, and lap recording functionality.

## Features

- Start Stopwatch
- Pause & Resume
- Reset Stopwatch
- Record Lap Times
- Responsive Design
- Clean & Modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Objective

To build an interactive stopwatch application capable of measuring time accurately while providing a simple and user-friendly interface.
